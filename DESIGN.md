# DESIGN.md

## 气质与意象

清晨大学图书馆的阅览桌——白色桌面、深色扶手椅、窗边透过来的冷光，桌上散落着论文与笔记本。学术的庄重感中透出前沿研究的锐利锋芒。不是冰冷的科技感，而是"知性沉淀后的突破"——像一篇 NeurIPS 最佳论文，表面克制，内里震撼。

## 配色方案

- **主色 (Primary)**: 深靛蓝 `oklch(0.35 0.15 260)` — 学术权威感，源自深夜台灯下的墨水蓝
- **辅色 (Accent)**: 琥珀橙 `oklch(0.72 0.17 55)` — 突破与灵感，源自论文被接收时的标记笔高亮
- **背景**: 纯白 `#FFFFFF` / 深色模式 `oklch(0.13 0.01 260)`
- **前景文字**: 近黑 `oklch(0.15 0 0)` / 深色模式 `oklch(0.95 0 0)`
- **Muted**: 暖灰 `oklch(0.96 0.005 260)` — 卡片与区域分隔

## 字体排版

- **标题字体**: Inter（Google Fonts）— 几何感强、适合学术标题的现代无衬线体
- **正文字体**: 系统字体栈（PingFang SC / Segoe UI / sans-serif）
- **排版节奏**: 标题大而疏、正文紧凑。Hero 区标题用 `text-5xl/6xl font-bold tracking-tight`，章节标题 `text-3xl font-semibold`，正文 `text-base leading-relaxed`

## 视觉策略

- 大量留白，内容呼吸感强
- 卡片使用极细边框 + 微妙阴影，不用粗重描边
- 图像处理：人物照片用圆角矩形裁切，论文封面保持原始比例
- 图标使用 Lucide Icons，线条 1.5px，与整体细线美学统一

## 动效与交互

- **入场动画**: 元素从下方 20px 淡入，duration 0.5s，ease-out，stagger 0.1s
- **悬停**: 卡片上移 4px + 阴影加深，transition 0.2s ease
- **页面切换**: 无硬跳转，使用 Framer Motion 的 fadeIn
- **导航**: 顶部固定，滚动时添加微妙底部阴影
- **禁忌**: 禁止弹跳动画、禁止旋转、禁止脉冲闪烁

## 页面结构

- **Hero**: 全宽，左侧标题+描述+按钮，右侧可放抽象装饰图形
- **Research Areas**: 3-4 列卡片网格，每张卡片图标+标题+简述
- **Publications**: 3 列响应式网格，卡片含封面缩略图+标题+作者+会议标签
- **People**: 按 role 分组，圆形/圆角矩形头像 + 姓名 + 职位
- **News**: 时间线式列表，左侧日期，右侧内容

## 设计禁忌

- 禁止科技蓝+紫渐变（过于模板化）
- 禁止全屏深色 Hero（学术站点保持清爽）
- 禁止使用粗重阴影（`shadow-lg` 以上）
- 禁止卡片使用圆角超过 12px（保持学术克制）
- 禁止使用 emoji 作为图标
